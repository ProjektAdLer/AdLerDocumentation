# Aufgabe erstellen

## Definition:

Falls ein Adaptivitätselement existiert, muss das Autorentool Lehrenden die Möglichkeit bieten, eine Aufgabe innerhalb
dieses Adaptivitätselements zu erstellen.

## Akzeptanzkriterien:

- Die Aufgabe muss [](AWA9010.md)erfüllen.
