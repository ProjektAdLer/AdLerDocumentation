# Lernwelt Übersicht Laden


# Definition:

Die Engine muss Lernenden die Möglichkeit bieten, eine Übersicht der verfügbaren Lernwelten zu laden.

# Akzeptanzkriterien:

- Benötigte Informationen sind mindestens ID und Name der Welten
