# Lernwelt laden


# Definition:

Die Engine muss Lernenden die Möglichkeit bieten, eine Lernwelt zu laden.

# Akzeptanzkriterien:

- Lernwelten sind immer Teil einer Welt (vgl. [Lernraum Laden](ELG0004.md))
