# Frage erstellen

## Definition:

Falls ein Adaptivitätselement mit einer Aufgabe existiert, muss das Autorentool Lehrenden die Möglichkeit bieten, eine
Frage innerhalb dieser Aufgabe zu erstellen.

## Akzeptanzkriterien:

- Die Frage muss [](AWA9012.md)erfüllen.
