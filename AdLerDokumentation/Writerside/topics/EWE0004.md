# Adaptivitätselement: Text als Hinweis anzeigen


## Definition:

Falls der [Hinweis](Adaptivitätshinweis-GE.md) des Adaptivitätselements ein Text ist, muss die Engine den Text anzeigen.

## Akzeptanzkriterien:

- Dem Lernenden wird der Text angezeigt
