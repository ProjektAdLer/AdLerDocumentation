# Weltverfügbarkeit prüfen


# Definition:
Die Engine muss prüfen können, ob eine spezifische Welt für den Lernenden verfügbar ist.


# Akzeptanzkriterien:
- Falls die angefragte Welt für den User nicht verfügbar ist, wird ein Fehler geworfen.
