# Lernraum laden


# Definition:

Die Engine muss Lernenden die Möglichkeit bieten, einen Lernraum zu laden.

# Akzeptanzkriterien:

- Lernräume sind immer Teil einer Lernwelt (vgl. [Lernwelt Laden](ELG0005.md))
